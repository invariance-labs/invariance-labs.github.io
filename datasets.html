SLM-COLLAPSE Benchmark Dataset

2,800 procedurally generated benchmark items across four task families — composition (chained multi-step reasoning, 1–8 hops), constraints (simultaneous instruction-following, 1–10 constraints), entity tracking (sequential state updates across a simulated data structure), and counting (character-level precision tasks). Every item is generated and graded by deterministic code rather than human annotation — the ground-truth answer is produced as a byproduct of constructing each question, and independently re-derived by a separate verification script across all 45,872 graded responses, with zero mismatches.
